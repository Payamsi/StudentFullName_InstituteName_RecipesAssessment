# 🍴 Recipes API 

A simple backend for storing and querying recipes.

#Features
- Load recipes from a JSON file into SQLite
- REST API with:
  - `/api/recipes` → pagination, sorted by rating
  - `/api/recipes/search` → search by title, cuisine, rating, total time
